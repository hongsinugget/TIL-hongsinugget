# 1일차 파이썬 공부

## 오늘의 배움!
- 파이썬은 대소문자를 구분한다
- 지금까지 #을 사용하여 한줄로 주석을 작성했는데 여러줄을 쓰고싶다면 독스트링(""")을 사용해야된다 (주의:들여쓰기 필수)
- 특히 문자열 출력을 할때 띄어쓰기는 안에서도 쓸수있고 콤마(,)를 쓸수있다
- 출력에서 문자열로 큰따옴표를 출력하려면 작은따옴표를 써야된다
- 출력할 따옴표를 구분하려면 \ 역슬래시를 앞에 붙여 문자열을 구분해줘야된다
- 맥북에서는 백틱을 사용하려면 옵션+` 백틱 키를 눌러야된다
- 역슬래시를 나오게 하고싶을땐 \\로 역슬래시를 두번써야된다
- 줄바꿈을 하려면 \n 을 사용한다
- input()을 사용하면 키보드에서 입력한 값을 출력한다
- 변수는 어떤 값을 저장할수있고 영어나 _ 또는 숫자 사용가능하다
- 정수는 소숫점이 없는 수 int를 사용한다
- "="는 대입연산자로 오른쪽 값을 왼쪽에 저장하라는 의미다
