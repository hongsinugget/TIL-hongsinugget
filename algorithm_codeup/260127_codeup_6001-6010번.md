# 2026.01.27 CodeUp 기초 100제 (6001~6010번)

## 6001번 : 기본 출력
- **문제:** `Hello`를 출력하시오.
- **코드:**
```python
print("Hello")
```

## 6002번 : 공백 포함 출력
- **문제:** `Hello World` (공백 포함) 출력.
- **코드:**
```python
print("Hello World")
```

## 6003번 : 줄 바꿈 출력
- **문제:** 두 줄에 걸쳐 출력.
- **코드:**
```python
print("Hello")
print("World")
```

## 6004번 : 작은따옴표(') 출력
- **문제:** `'Hello'` 출력.
- **코드:**
```python
print("'Hello'")
```

## 6005번 : 큰따옴표(") 출력
- **문제:** `"Hello World"` 출력.
- **코드:**
```python
print('"Hello World"')
```

## 6006번 : 특수문자 출력 (이스케이프 문자)
- **문제:** `"!@#$%^&*()'` 출력.
- **코드:**
```python
print("\"!@#$%^&*()'")
```

## 6007번 : 파일 경로 출력 (역슬래시)
- **문제:** `C:\Download\'hello'.py` 형식 출력.
- **코드:**
```python
print("\"C:\\Download\\'hello'.py\"")
```

## 6008번 : 출력문 자체를 출력하기
- **문제:** `print("Hello\nWorld")` 라는 문장 그대로 출력.
- **코드:**
```python
print('print("Hello\\nWorld")')
```

## 6009번 : 문자 1개 입력받아 출력
- **문제:** 변수에 문자 저장 후 출력.
- **코드:**
```python
c = input()
print(c)
```

## 6010번 : 정수 입력받아 출력
- **문제:** 입력받은 값을 정수(`int`)로 변환하여 출력.
- **코드:**
```python
n = input()
n = int(n)
print(n)
```
