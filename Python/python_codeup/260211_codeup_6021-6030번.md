## 6021 : 단어 1개 입력받아 나누어 출력하기
- **문제**  각 문자를 한 줄에 한 문자씩 줄을 바꿔 출력한다.
- **내 풀이** 0부터 시작한다
- **코드**
```python
s = input()
print(s[0])
print(s[1])
print(s[2])
print(s[3])
print(s[4])
```

## 6022 : 연월일 입력받아 나누어 출력하기
- **문제**  년도(YY) 월(MM) 일(DD)을 공백으로 구분해 한 줄로 출력한다.
- **내 실수** split를 쓰는 문제인줄알았는데 문자열 슬라이싱 문제였다. 끝번호를 가져오는게 아니라 끝번호 바로 앞에 있는 숫자를 가져오는것
- **코드**
```python
s = input()
yy = s[0:2]
mm = s[2:4]
dd = s[4:6]
print(yy, mm, dd)
```

## 6023 : 시분초 입력받아 분만 출력하기
- **문제**  분만 출력한다.
- **내 실수** 한자리수의 시간으로 형식이 달라질경우 슬라이싱 실수가 생길수있다. 이럴때 split 함수를 써야함으로 : 기호를 써준다. split 함수를 쓰게되면 리스트로 들어가지게되고 값의 길이에 상관없이 들어가게됨.
- **코드**
```python
s = input().split(":")
m = s[1]
print(m)
```

## 6024 : 단어 2개 입력받아 이어 붙이기
- **문제**  입력된 2개의 단어를 순서대로 붙여 출력한다.
- **코드**
```python
w1, w2 = input().split()
s = w1 + w2
print(s)
```

## 6025 : 정수 2개 입력받아 합 계산하기
- **문제**  두 정수의 합을 출력한다.
- **내 실수** int 변환에서 마이너스를 했는데 더한값이 나왔다. 마이너스는 마이너스끼리 하면 +가 됨
- **코드**
```python
a, b = input().split()
c = int(a) + int(b)
print(c)
```

## 6026 : 실수 2개 입력받아 합 계산하기
- **문제**  두 실수의 합을 출력한다.
- **내 실수** input과 split에서는 입력값을 한줄로 쓰면 두개를 구분해서 사용할수있는데 줄바꿈 입력값일경우 input 값을 여러번 써야된다.
- **코드**
```python
a = input()
b = input()
c = float(a) + float(b)
print(c)
```

## 6027 : 10진 정수 입력받아 16진수로 출력하기
- **문제**  16진수(소문자) 형태로 출력한다.
- **내 풀이** %는 뒤에있는 값을 채워넣어라는 뜻이고 255를 16으로 나누면 15의 값과 나머지 15가 된다 
- **코드**
```python
a = input()
b = int(a)
print("%x" % b)
```
## 6028 : 10진 정수 입력받아 16진수로 출력하기
- **문제**  16진수(대문자) 형태로 출력한다.
- **코드**
```python
a = input()
b = int(a)
print("%X" % b)
```

## 6029 : 16진 정수 입력받아 8진수로 출력하기
- **문제**  8진수 형태로 출력한다.
- **코드**
```python
a = input()
b = int(a, 16)

```

## 6030 : 영문자 1개 입력받아 10진수로 변환하기
- **문제**  유니코드 값을 10진수로 출력한다.
- **내 풀이** ord(input())은 입력받은 문자를 10진수 유니코드로 변환한다
- **코드**
```python
n = ord(input())
print(n)
```
