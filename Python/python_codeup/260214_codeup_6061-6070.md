## 6061 : 비트단위로 OR 하여 출력하기
- **문제**  두 정수를 비트단위(bitwise)로 or 계산을 수행한 결과를 10진수로 출력한다.
- **코드**
```python
a, b = map(int, input().split())
print(a | b)
```

## 6062 : 비트단위로 XOR 하여 출력하기
- **문제**  두 정수를 비트단위(bitwise)로 xor 계산을 수행한 결과를 10진수로 출력한다.
- **코드**
```python
a, b = map(int, input().split())
print(a ^ b)
```

## 6063 : 정수 2개 입력받아 큰 값 출력하기
- **문제**  두 정수를 비트단위(bitwise)로 xor 계산을 수행한 결과를 10진수로 출력한다.
- **코드**
```python
a, b = map(int, input().split())
print(a if (a >= b) else b)
```

## 6064 : 정수 3개 입력받아 가장 작은 값 출력하기
- **문제**  가장 작은 값을 출력한다.
- **코드**
```python
a, b, c = map(int, input().split())
print((a if a < b else b) if ((a if a < b else b) < c) else c)
```

## 6065 : 정수 3개 입력받아 짝수만 출력하기
- **문제**  짝수만 순서대로 줄을 바꿔 출력한다.
- **코드**
```python
a, b, c = map(int, input().split())
if a % 2 == 0:
    print(a)
if b % 2 == 0:
    print(b)
if c % 2 == 0:
    print(c)
```

## 6066 : 정수 3개 입력받아 짝/홀 출력하기
- **문제**  입력된 순서대로 짝(even)/홀(odd)을 줄을 바꿔 출력한다.

```python
a, b, c = map(int, input().split())
if a % 2 == 0:
    print("even")
else:
    print("odd")
if b % 2 == 0:
    print("even")
else:
    print("odd")
if c % 2 == 0:
    print("even")
else:
    print("odd")
```

## 6067 : 정수 1개 입력받아 분류하기
- **문제**  음수이면서 짝수이면, A
음수이면서 홀수이면, B
양수이면서 짝수이면, C
양수이면서 홀수이면, D
를 출력한다.
- **코드**
```python
n = int(input())
if n < 0:
    if n % 2 == 0:
        print("A")
    else:
        print("B")
else:
    if n % 2 == 0:
        print("C")
    else:
        print("D")
```

## 6070 : 월 입력받아 계절 출력하기
- **문제**  계절 이름을 출력한다.
- **코드**
```python
a = int(input())
if a // 3 == 1:
    print("spring")
elif a // 3 == 2:
    print("summer")
elif a // 3 == 3:
```
