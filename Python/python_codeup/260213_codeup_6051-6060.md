## 6051 : 정수 2개 입력받아 비교하기4
- **문제**  a와 b가 다른 경우 True 를, 그렇지 않은 경우 False 를 출력한다.
- **내 풀이** 비교/관계 연산자중에 !=는 왼쪽값이 오른쪽과 다를때 true /false를 계산
- **코드**
```python
a, b = map(int, input().split())
print(a != b)
```

## 6052 : 정수 입력받아 참 거짓 평가하기
- **문제**  입력된 값이 0이면 False, 0이 아니면 True 를 출력한다.
- **코드**
```python
a = int(input())
print(bool(a))
```

## 6053 : 참 거짓 바꾸기
- **문제**  입력된 정수의 불 값이 False 이면 True, True 이면 False 를 출력한다.
- **내 풀이** 겹쳐 작성하면 한번에 한단계 순서대로 처리된다.
- **코드**
```python
a = bool(int(input()))
print(not a)
```

## 6054 : 둘 다 참일 경우만 참 출력하기
- **문제**  둘 다 True 일 경우에만 True 를 출력하고, 그 외의 경우에는 False 를 출력한다
- **코드**
```python
a, b = map(int, input().split())
print(bool(a and b))
```

## 6055 : 하나라도 참이면 참 출력하기
- **문제**  하나라도 참일 경우 True 를 출력하고, 그 외의 경우에는 False 를 출력한다.
- **코드**
```python
a, b = map(int, input().split())
print(bool(a or b))
```

## 6056 : 하나라도 참이면 참 출력하기
- **문제**  두 값의 True / False 값이 서로 다를 경우만 True 를 출력하고, 그 외의 경우에는 False 를 출력한다.
- **코드**
```python
a, b = map(int, input().split())
print(bool(a and not (b)) or bool(not (a) and b))
```

## 6057 :  참/거짓이 서로 같을 때에만 참 출력하기
- **문제** 두 값의 True / False 값이 서로 같을 경우만 True 를 출력하고, 그 외의 경우에는 False 를 출력한다.
- **코드**
```python
a, b = map(int, input().split())
print(bool(a == b))
```

## 6058 :  참둘 다 거짓일 경우만 참 출력하기
- **문제** 두 값의 True / False 값이 모두 False 일 때만 True 를 출력하고, 그 외의 경우에는 False 를 출력한다.
- **코드**
```python
a, b = map(int, input().split())
print(bool(not (a) and not (b)))
```
## 6059 :  비트단위로 NOT 하여 출력하기
- **문제** 비트 단위로 1 -> 0, 0 -> 1로 바꾼 후 그 값을 10진수로 출력한다.
- **코드**
```python
a = int(input())
print(~a)
```

## 6060 :  비트단위로 AND 하여 출력하기
- **문제** 두 정수를 비트단위(bitwise)로 and 계산을 수행한 결과를 10진수로 출력한다.
- **코드**
```python
a, b = map(int, input().split())
print(a & b)
```
