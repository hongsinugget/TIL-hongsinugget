## 6031 : 정수 입력받아 유니코드 문자로 변환하기
- **문제**  유니코드 문자로 출력한다.
- **내 풀이** chr()은 정수값에서 문자 유니코드로 바꿔줌
- **코드**
```python
 c = int(input())
print(chr(c))
```

## 6032 : 정수 1개 입력받아 부호 바꾸기
- **문제**  부호를 바꿔 출력한다.
- **내 풀이** 입력한값을 정수로 변환하고 -1을 받으면 플러스가 된다
- **코드**
```python
n = int(input())
print(-n)
```
## 6033 : 문자 1개 입력받아 다음 문자 출력하기
- **문제**  그 다음 문자를 출력한다.
- **내 풀이** 문자를 숫자로 바꿔주고 숫자를 문자로 한번 더 바꿔줘야된다
- **코드**
```python
n = ord(input())
print(chr(n + 1))
```

## 6034 : 정수 2개 입력받아 차 계산하기
- **문제**  첫 번째 정수에서 두 번째 정수를 뺀 차를 출력한다.
- **내 실수** split은 리스트로 들어가기 때문에 문자로 인식해서 합이되지않는다 여기서 int 함수로 변환해줘야된다
- **코드**
```python
a, b = input().split()
c = int(a) - int(b)
print(c)
```

## 6041 : 정수 2개 입력받아 나눈 나머지 계산하기
- **문제**  a를 b로 나눈 나머지를 출력한다.
- **내 풀이** 각각 int를 번거롭게 작성하기 어려우니 map을 통해 한번에 묶어 정수형으로 변환한다.
- **코드**
```python
a, b = map(int, input().split())
print(a % b)
```

## 6042 : 실수 1개 입력받아 소숫점이하 자리 변환하기
- **문제**  소숫점 이하 두 번째 자리까지의 정확도로 반올림한 값을 출력한다.
- **코드**
```python
a = input()

print(format(b, ".2f"))
```
## 6043 : 실수 2개 입력받아 나눈 결과 계산하기
- **문제**  f1을 f2로 나눈 결과를 소숫점 이하 넷째 자리에서 반올림하여 소숫점 세 번째 자리까지 출력한다.

- **코드**
```python
a, b = map(float, input().split())
c = a / b
print(format(c, ".3f"))
```

## 6044 : 정수 2개 입력받아 자동 계산하기
- **문제**  첫 번째 줄에 합
두 번째 줄에 차,
세 번째 줄에 곱,
네 번째 줄에 몫,
다섯 번째 줄에 나머지,
여섯 번째 줄에 나눈 값을 순서대로 출력한다.
(실수, 소수점 이하 둘째 자리까지의 정확도로 출력)

- **코드**
```python
a, b = map(int, input().split())
print(a + b)
print(a - b)
print(a * b)
print(a // b)
print(a % b)
print(format(a / b, ".2f"))
```
## 6045 : 정수 3개 입력받아 합과 평균 출력하기
- **문제**  합과 평균을 공백을 두고 출력한다.
평균은 소숫점 이하 셋째 자리에서 반올림하여 둘째 자리까지 출력한다.
- **코드**
```python
a, b, c = map(int, input().split())
d = a + b + c
print(d, format(d / 3, ".2f"))
```

## 6049 : 정수 2개 입력받아 비교하기2
- **문제** a와 b의 값이 같은 경우 True 를, 그렇지 않은 경우 False 를 출력한다.
- **내 풀이** 왼쪽의 결과가 오른쪽이랑 같으면 ==을 사용함
- **코드**
```python
a, b = map(int, input().split())
print(a == b)
```

## 6050 : 정수 2개 입력받아 비교하기3
- **문제** b가 a보다 크거나 같은 경우 True 를, 그렇지 않은 경우 False 를 출력한다.
- **내 풀이** 비교연산자인 <=는 왼쪽의 값이 오른쪽값보다 작거나 같을때 사용
- **코드**
```python
a, b = map(int, input().split())
print(a <= b)
```
