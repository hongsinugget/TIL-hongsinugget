## 6031 : 정수 입력받아 유니코드 문자로 변환하기
- **문제**  유니코드 문자로 출력한다.
- **내 풀이** chr()은 정수값에서 문자 유니코드로 바꿔줌
- **코드**
```python
 c = int(input())
print(chr(c))
```

## 6032 : 정수 1개 입력받아 부호 바꾸기
- **문제**  부호를 바꿔 출력한다.
- **내 풀이** 입력한값을 정수로 변환하고 -1을 받으면 플러스가 된다
- **코드**
```python
n = int(input())
print(-n)
```
## 6033 : 문자 1개 입력받아 다음 문자 출력하기
- **문제**  그 다음 문자를 출력한다.
- **내 풀이** 문자를 숫자로 바꿔주고 숫자를 문자로 한번 더 바꿔줘야된다
- **코드**
```python
n = ord(input())
print(chr(n + 1))
```

## 6034 : 정수 2개 입력받아 차 계산하기
- **문제**  첫 번째 정수에서 두 번째 정수를 뺀 차를 출력한다.
- **내 실수** split은 리스트로 들어가기 때문에 문자로 인식해서 합이되지않는다 여기서 int 함수로 변환해줘야된다
- **코드**
```python
a, b = input().split()
c = int(a) - int(b)
print(c)
```

## 6035 : 실수 2개 입력받아 곱 계산하기
- **문제**  첫 번째 실수와 두 번째 실수를 곱한 값을 출력한다.
- **코드**
```python
a, b = input().split()
c = float(a) * float(b)
print(c)
```

## 6036 : 단어 여러 번 출력하기
- **문제**  입력된 단어를 입력된 횟수만큼 반복해 출력한다.
- **코드**
```python
w, n = input().split()
print(w * int(n))
```

## 6037 : 문장 여러 번 출력하기
- **문제**  입력된 횟수만큼 입력된 문장을 출력한다.
- **코드**
```python
w = input()
l = input()
print(int(w) * l)
```

## 6038 : 정수 2개 입력받아 거듭제곱 계산하기
- **문제**  a를 b번 거듭제곱한 값을 출력한다.
- **코드**
```python
a, b = input().split()
```

## 6040 : 정수 2개 입력받아 나눈 몫 계산하기
- **문제**  a를 b로 나눈 몫을 출력한다.
