# 5일차 파이썬 공부

## 오늘의 배움!

- bool을 이용하면 입력된 식이나 값을 평가해 불 형의(true/false) 값을 출력해준다
- 파이썬에서는 정수값 0은 false(거짓)으로 평가됨 그외는 true
- 참 또는 거짓의 논리값을 표현할때 not연산을 사용
- 거짓의 논리값 불의 값을 다룰땐 not,and,or이 있다
- and 예약어는 두 불의 값이 모두 true일때 true로 계산
- or 예약어는 주어진 두 불 값중에서 하나라ㅗㄷ true면 true이고 나머지는 false 거짓으로 된다
- 참 거짓이 서로 다를때에만 true로 계산하는 논리 연산을 XOR 연산이라고 부른다
- 논리연산자는 사칙 연산자처럼 여러번 중복해서 쓸수있다
- 비트 단위 연산자는 (~.&,|,^,<<,>>)가 있다
- 비트 단위 and 연산은 두 비트열이 주어질때 둘다 1인부분의 자리만 1로만들어줌
