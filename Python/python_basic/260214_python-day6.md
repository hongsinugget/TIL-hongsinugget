# 6일차 파이썬 공부

## 오늘의 배움!

- | 는 파이프 연산자라고 불린다.
- xor은 비트단위연산자에서 ^를 사용한다
- 3항 연산자 if,else가 있는데 ex: "x if C else y"가 있다면 C는 true or false를 평가할 조건식또는 값이고 x,는 C의 평가 결과에 따라  True,y는 C의 평가 결과에 따라 false가 된다
- 순서에 따라 단계별로 하나씩 수행
- 논리적으로 한 단위씩 처리 해야되는경우 콜론(:)을 찍고 들여쓰기로 작성
- else는 if없이 혼자 사용되지않는다
- 파이썬에서는 들여쓰기가 중요하다 코드블럭을 구분하기 때문
