# 4일차 파이썬 공부

## 오늘의 배움!

- ord()는 문자에서 정수값 형태로 바꿔주고 chr()은 정수값을 문자로 바꿈
- *은 곱셈이다
- **은 거듭제곱을 함
- // 나눈 몫
- 나눈 나머지를 계산하는 연산자를 %로 표현한다
예시로 a%b와 같이 작성하면 a를 b로 나눈 나머지를 계산한다.
- 일일히 int를 넣기엔 번거러우니 map이란 함수를 사용할수있고 map은 똑같은 함수를 한꺼번에 적용할수있게 묶어주는 역할이다.
- format(수,".2f")를 사용하면 소숫점 두번째자리까지만 보여줌 세번째 자리에서 반올림이 된다.
- 2배를 만들때 *2도 되지만 비트단위시프트 연산자인 <<>>를 사용해서 2배로 곱할수있다
- 컴퓨터 내부에는 2진수 형태로 저장되기 때문에 2진수 형태로 저장되어있는 값을 지정한 비트수만큼 밀어내면 2배 혹은 1/2로 할수있다 
- a<<2를 하면 a 2에2승이 되는데 "a<<b"를 하게되면 a*2에n승이된다
- 어떤값을 비교하기 위해쓰는 연산자 비교 연산자라고 한다
- 비교/관계연산자도 사칙연산을 수행하고 참또는 거짓으로 계산해주는 연산자이다(<,>,<=,>=,==(같다),!=(다르다))
- 참또는 거짓을 표현하는 저장값은 불(bool)불리언(boolean)이라고 한다
