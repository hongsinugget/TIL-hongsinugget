# 2일차 파이썬 공부

## 오늘의 배움!

- 숫자와 소숫점을 사용해 표현한 수를 (float) 실수라고 한다
- 입력한 값을 원하는 형태로 계산하거나 처리하기 위해서는 입력한 값이 어떤 데이터인지 (정수,실수,문자열)인지 구분해주어야 한다
- 코드 사이에 주석을 넣고 싶을때 (#)을 사용한다
- split()은 공백을 기준으로 입력된 값을 나누어 자른다
- split(':') 스플릿 안에 ':'기호를 넣으면 :기준으로 자른다
- print안에 sep=':'를 사용하면 ':'기호를 사이에 두고 값을 출력한다. (sep은 분류기호를 의미)
