# 3일차 파이썬 공부

## 오늘의 배움!

- 문자열은 인덱스 0번부터 순서가 시작된다!
- [a:b]를 할 경우 단어에서 a번째문자에서 b-1문자까지 잘라내서 출력한다 (끝번호 바로 앞까지 가져옴)
- split은 구분 뿐만아니라 리스트로 안전하게 넣을수있다
- 문자들로 구성된 문장을 문자열(string)이라고 부른다
- 문자열+문자열은 합칠수있다
- 문자열로 되어있는 수는 정수로 바꾸기 위해서는 int()를 사용할수있다
- 10진법은 한자리에 10개의 문자를 사용하고 16진법은 영문 소문자를 사용하는 경우에 한자리에 16개의 문자를 사용한다
- %x로 출력하면 16진수 소문자로 출력된다(대문자로 할경우 대문자 X 사용)
- %o를 사용하면 8진수 출력
- ord()는  어떤 문자의 순서를 위치한다 (ex:ord(c) : 문자 c 를 10진수로 변환한 값 )
- 유니코드는 세계 여러나라의 문자를 공통된 코드의 값을 저장할때 사용하는 표준코드
