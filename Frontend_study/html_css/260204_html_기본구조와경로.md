# 📅 2026.02.04 HTML 기본 구조와 경로 학습

## 1. HTML 기본 세팅 (Boilerplate)
- **단축키 `!` + `Tab`**: VS Code에서 HTML5 기본 구조를 자동으로 생성해준다.
- `DOCTYPE html`: 현재 문서가 **HTML5** 버전임을 브라우저에 알려주는 선언.

## 2. `<head>` 태그 : 문서의 정보와 설정
`<head>` 안에는 눈에 보이지 않는 설정값들이 들어간다.

### 메타 태그 (`<meta>`)
정보(Data)를 설명하는 데이터. 닫는 태그가 없다.
- `charset="UTF-8"`: 문자 인코딩 방식. 이 설정이 없으면 한글("김")이 깨져서("ㄱ ㅣ ㅁ") 보일 수 있다.
- `name="viewport"`: 모바일 화면 설정.
  - `content="width=device-width"`: 화면 너비를 디바이스 너비에 맞춘다. (이게 없으면 모바일에서 글씨가 개미만하게 보임)

### 스타일 연결 (`<link>`)
CSS 파일을 가져올 때 사용한다.
```
<link rel="stylesheet" href="./main.css" />
```
- **rel** (Relationship): 현재 문서와 가져올 파일의 관계 (stylesheet = 스타일 시트).

- **href** (Hypertext Reference): 가져올 파일의 경로.

- 참고: CSS는 <head> 안에 <style>...</style> 태그로 직접 쓸 수도 있지만, 보통은 link로 외부 파일을 가져온다.

## 3. `<body>` 태그 : 눈에 보이는 내용과 스크립트

자바스크립트 연결 (`<script>`)
- **src** (Source): JS 파일의 경로를 지정한다.

- **이미지** (`<img>`)
alt (Alternative Text): 이미지가 깨지거나 로딩되지 않았을 때 대신 보여줄 텍스트. (시각장애인을 위한 스크린 리더가 이 내용을 읽어줌)

- **링크** (`<a>`)
href: 이동할 문서의 경로. 텍스트를 클릭하면 해당 주소로 이동한다.

## 4. 상대경로 VS 절대경로
1) **상대 경로**
내 위치(현재 파일)를 기준으로 파일을 찾는다.

- ./ : 현재 폴더 (주변). 생략 가능하다. (예: main.css = ./main.css)

- ../ : 상위 폴더 (부모). 밖으로 한 번 나간다.

2) **절대 경로**
내 위치와 상관없이 고정된 주소다.

- URL: http, https로 시작하는 웹사이트 주소.

- 최상위 루트(/): 내 컴퓨터(또는 서버)의 가장 뿌리가 되는 폴더부터 시작하는 경로. (예: /users/admin/desktop/...)
